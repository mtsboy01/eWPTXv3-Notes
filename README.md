# 📚 eWPTXv3 Study Notes & Methodology

> Comprehensive notes, methodology, and resources for the 
> eLearnSecurity Web Application Penetration Tester eXtreme (eWPTXv3) certification.

## 🎯 About This Repo

This repository contains my personal study notes, pentest methodology, 
checklists, and resources compiled while preparing for the eWPTXv3 exam.
Useful for security researchers and penetration testers working on 
advanced web application security.

|-------|-------------|

| Recon & Enumeration | Information gathering techniques |
| Authentication Attacks | Login bypass, brute force, JWT attacks |
| XSS (Cross-Site Scripting) | Reflected, Stored, DOM-based |
| SQL Injection | Manual & automated techniques |
| SSRF | Server-Side Request Forgery |
